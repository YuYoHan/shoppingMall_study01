# ShoppingMall

여기 레포지토리는 `스프링 부트 쇼핑몰 프로젝트 with JPA` 책을 공부한 코드입니다. REST 방식으로 변경해서 만들 생각입니다.<br />


## 목차
1장 : 개발환경 구축 <br/>
2장 : Spring Data JPA <br/>
3장 : 스프링 시큐리티를 이용한 회원 가입 및 로그인 <br/>
4장 : 연관 관계 매핑 <br/>
5장 : 상품 등록 및 조회 <br/>
6장 : 주문<br/>
7장 : 장바구니

## 사용기술
1. SpringBoot
2. JPA
3. MySQL
4. SpringSecurity
5. JWT
